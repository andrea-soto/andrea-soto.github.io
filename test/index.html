Bazaar, a tool to research open source projects

You’ve tried that open source project that everyone is talking about and you are convinced that it is the best tool for the job. But how to get buy-in from other stakeholders in your company? Bazaar to the rescue!

We developed a set of data-driven metrics about open source projects that will help you determine if a project is stable, if it has a growing community that supports it, and if development is likely to continue. Our unbiased metrics will help you and your team decide whether that open source project is the right choice.

Don’t have an open source project in mind? No problem, explore our dashboard to find projects and see how they compare to others. You might even discover a great project and be the first one to start talking about it.
