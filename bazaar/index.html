Bazaar

You’ve tried that open source project that everyone is talking about and you are convinced that it is exactly what you need. But how to get buy-in from other stakeholders in your company? Bazaar to the rescue! Bazaar is a data-driven tool to research open source projects.

We developed a set of data-driven metrics about open source projects that will help you determine if a project is stable, if it has a growing community that supports it, and if development is likely to continue. Our unbiased metrics will help you and your team decide whether that open source project is the right choice or not.

Don’t have an open source project in mind? No problem, explore our dashboard to find projects and see how they compare to others. You might discover the next great project and become the first one to talk about it.

When discussing whether to use open source software in production settings, one of the first topics to come up is software support. Because you don't have a service agreement that guarantees ongoing support, it is important to know in advance if the project has a good track record of addressing problems and fixing bugs.
